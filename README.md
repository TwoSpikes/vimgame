# What is this?

This is repository with some games that I ported to Vimscript (to Vim/NeoVim)

# Snake

## Quick start

```console
$ vim ./snake.vim
```

In Vim/NeoVim run:

```
:source %
:call Snake()
```

# Spinning donut (like donut.c)

## Quick start

```console
$ vim ./donut.vim
```

In Vim/NeoVim run:

```
:source %
:call Donut()
```
